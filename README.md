# Mardini IA — Design System

**Mardini IA** es una agencia de automatización con inteligencia artificial para PyMEs latinoamericanas. Ofrecemos workflows de n8n, agentes de voz, extractores de leads y sistemas de automatización de ventas.

## Stack

- React 18 + Vite + Tailwind CSS
- Framer Motion para animaciones
- Radix UI para componentes accesibles
- GoHighLevel (GHL) para formularios y CRM
- Deploy en Vercel

## Identidad Visual

- **Tema:** 100% dark (nunca light)
- **Primario:** Azul (#3b82f6) con gradientes hacia azul oscuro y índigo
- **Fondo base:** #0a0a0f (casi negro)
- **Estética:** Glassmorphism + glow effects + animaciones sutiles
- **Referencia visual:** Stripe Sessions 2026 — secciones oscuras, canvas gradients, tipografía limpia

## Categorías de negocio (4 áreas con gradientes propios)

1. **Generación de Ingresos** → verde-esmeralda (`#22c55e → #10b981`)
2. **Optimización de Costos** → amarillo-naranja (`#eab308 → #ea580c`)
3. **Recuperación de Tiempo** → azul-cyan (`#3b82f6 → #06b6d4`)
4. **Diferenciación Premium** → púrpura-rosa (`#a855f7 → #db2777`)

## Principios

- Mobile-first siempre
- Glassmorphism: backdrop-blur + fondos semitransparentes
- Animaciones que mejoran sin distraer
- WCAG AA en todos los colores
- Spacing consistente con escala Tailwind
